# csc125-final-project

##Project Description
This project is a Beauty Parlor Booking System that uses dictionaries, functions, loops, and input/output to manage appointments. The program follows the pseudocode to organize logic and ensure proper functionality.

##Features
Dictionary storing beauty parlor services and their prices
Custom function book_appointment() that allows users to schedule an appointment
While loop that continuously runs the menu until the user exits
User input with basic validation using try/except
Formatted output displaying booking details clearly
